# Raw data files

The following files from
[https://apps.neh.gov/open/data/](https://apps.neh.gov/open/data/) will go into
this subdirectory:

- `NEH_Grants1960s.csv`
- `NEH_Grants1970s.csv`
- `NEH_Grants1980s.csv`
- `NEH_Grants1990s.csv`
- `NEH_Grants2000s.csv`
- `NEH_Grants2010s.csv`
- `NEH_Grants2020s.csv`

`make_data.R` will download these files if not found in this subdirectory.
